## 5.1 图像退化模型

### 5.1.1 基本概念

**图像退化**：图像在获取、传输、存储过程中，由于多种因素导致图像质量下降。

**图像复原**：从退化图像中恢复原始图像，通过估计退化模型并进行逆操作。

### 5.1.2 退化模型

```
原始图像 f(x,y) → 退化函数 H → 加性噪声 η(x,y) → 退化图像 g(x,y)
```

**数学表达式**：
```
g(x,y) = H[f(x,y)] + η(x,y)
```

**频率域表示**：
```
G(u,v) = H(u,v) × F(u,v) + N(u,v)
```

### 5.1.3 退化模型的性质

| 性质 | 说明 |
|------|------|
| **线性** | H[af₁ + bf₂] = aH[f₁] + bH[f₂] |
| **位置不变性** | H[f(x-α, y-β)] = g(x-α, y-β) |
| **加性** | 输出 = 真实图像 + 噪声 |

### 5.1.4 常见退化原因

1. **运动模糊**：拍摄时相机或物体运动
2. **大气湍流**：遥感图像的大气散射
3. **镜头散焦**：光学系统焦点偏移
4. **噪声干扰**：电子传感器噪声
5. **几何畸变**：镜头畸变、投影变换

## 5.2 噪声模型

### 5.2.1 噪声的类型

| 噪声类型 | 特征 | 典型来源 |
|----------|------|----------|
| 高斯噪声 | 钟形分布，均值μ，方差σ² | 电子电路、传感器噪声 |
| 均匀噪声 | 等概率分布 | 量化噪声 |
| 椒盐噪声 | 双峰，脉冲噪声 | 开关错误、尘粒 |
| 指数噪声 | 指数分布 | 激光噪声 |
| 伽马噪声 | 瑞利分布 | 雷达噪声 |

### 5.2.2 高斯噪声

**概率密度函数**：
```
p(z) = (1/√(2πσ²)) × e^(-(z-μ)²/2σ²)
```

**参数**：
- μ = 均值
- σ² = 方差

**特点**：
- 最常见的噪声模型
- 数学处理方便
- 中心极限定理支持

### 5.2.3 椒盐噪声

**概率密度函数**：
```
P(z) = Pa (z = a) + Pb (z = b)
```

**特点**：
- 又称脉冲噪声
- 通常 a = 最小灰度, b = 最大灰度
- **中值滤波效果最好**

### 5.2.4 噪声参数估计

**方法一：直方图估计**
- 从图像平坦区域估计噪声方差
- 假设噪声均值为0

**方法二：均值-方差估计**
- 在均匀区域计算局部均值和方差
- 估算噪声参数

## 5.3 空间域滤波复原

### 5.3.1 均值滤波器

| 滤波器 | 公式 | 特点 |
|--------|------|------|
| 算术均值 | ĝ = (1/mn)Σ f(x,y) | 模糊 |
| 几何均值 | ĝ = (Π f(x,y))^(1/mn) | 保留细节更好 |
| 谐波均值 | ĝ = mn/Σ(1/f(x,y)) | 对盐噪声有效 |
| 逆谐波均值 | ĝ = Σ f^(Q+1) / Σ f^Q | Q阶混合滤波 |

### 5.3.2 统计排序滤波器

**中值滤波器**：
- 对椒盐噪声非常有效
- 保护边缘
- 计算简单

**最大值/最小值滤波器**：
- 最大值：消除盐噪声
- 最小值：消除胡椒噪声

**自适应滤波器**：
- 根据局部特征调整滤波参数
- 效果更好但计算更复杂

## 5.4 频率域滤波复原

### 5.4.1 周期噪声复原

**陷波滤波器**：
- 针对特定频率的干扰
- 阻止或通过指定频率点

**带阻滤波器**：
- 阻止一定频率范围
- 用于消除周期性条纹

### 5.4.2 频率域去噪

**方法**：
1. 识别噪声频率成分
2. 设计陷波滤波器
3. 应用滤波器去除噪声

## 5.5 退化函数的估计

### 5.5.1 图像观察估计法

```
g(x,y)
  ↑
观察 | 退化模型
----- | --------
清晰区域 | 推测 H
  ↓
估计 Ĥ
```

**原理**：从图像中寻找可以作为退化分量的信息

### 5.5.2 试验估计法

使用相同的系统设置获取已知输入图像

```
已知输入 f(x,y) → 系统 H → 观测输出 g(x,y)
                              ↓
                       估计 H(u,v) = G(u,v)/F(u,v)
```

## 5.6 逆滤波与维纳滤波

### 5.6.1 逆滤波

**原理**：直接进行逆变换

```
F̂(u,v) = G(u,v) / H(u,v)
```

**问题**：
- 当 H(u,v) → 0 时，放大噪声
- 实际应用需要处理

**改进**：
```
F̂(u,v) = [H*(u,v)/|H(u,v)|²] × G(u,v) / H(u,v)
        = H*(u,v) × G(u,v) / |H(u,v)|²
```

### 5.6.2 最小均方误差滤波（维纳滤波）

**准则**：最小化均方误差

**滤波器**：
```
Ĥ(u,v) = H*(u,v) / [|H(u,v)|² + Sη(u,v)/Sf(u,v)]

其中：
  Sη(u,v) = 噪声功率谱
  Sf(u,v) = 图像功率谱
```

**简化形式**（假设噪声功率谱已知）：
```
Ĥ(u,v) = 1/H(u,v) × [|H(u,v)|² / (|H(u,v)|² + K)]

K = 常数（信噪比估计）
```

**优点**：
- 考虑噪声影响
- 比逆滤波更稳定
- 广泛使用

### 5.6.3 约束最小二乘滤波

**准则**：最小化拉普拉斯算子的范数

**滤波器**：
```
Ĥ(u,v) = H*(u,v) / [|H(u,v)|² + γ × |P(u,v)|²]

其中 P(u,v) 是拉普拉斯算子的傅里叶变换
γ 是常数
```

## 5.7 几何校正与几何变换

### 5.7.1 基本概念

**几何校正**：修正图像的几何畸变

**变换类型**：
- 刚性变换：平移、旋转
- 相似变换：缩放
- 仿射变换：平移、缩放、旋转、剪切
- 投影变换：透视变换

### 5.7.2 仿射变换

```
[x']   [a b] [x]   [e]
[y'] = [c d]×[y] + [f]

[x' y' 1] = [x y 1] × [a c 0
                      b d 0
                      e f 1]
```

**特殊情况**：
| 变换 | 矩阵 |
|------|------|
|恒等|[1 0 0; 0 1 0; 0 0 1]|
|平移|[1 0 0; 0 1 0; tx ty 1]|
|缩放|[sx 0 0; 0 sy 0; 0 0 1]|
|旋转|[cos -sin 0; sin cos 0; 0 0 1]|

### 5.7.3 灰度插值

**最近邻插值**：
- 取最近像素的值
- 简单快速
- 块效应明显

**双线性插值**：
- 用4个邻点线性组合
- 平滑效果
- 常用

**双三次插值**：
- 用16个邻点
- 边缘保持更好
- 计算量大

## 5.8 图像重建

### 5.8.1 基本概念

**目的**：从投影数据重建截面图像

**典型应用**：
- CT (计算机断层扫描)
- MRI (磁共振成像)
- 雷达干涉测量

### 5.8.2 投影定理

**中心切片定理**：
```
投影的傅里叶变换 = 二维傅里叶变换的一个切片

Pω(R) ⟷ F(u,v) |沿着 v = ωu 的切片
```

### 5.8.3 反投影法

**原理**：
1. 获取各个角度的投影
2. 将每个投影反投射回图像空间
3. 叠加所有反投影

**问题**：
- 产生星状伪影
- 需要滤波修正

### 5.8.4 滤波反投影法

**步骤**：
1. 对每个投影进行滤波（使用滤波器如 Ram-Lak）
2. 将滤波后的投影反投影
3. 叠加所有反投影

**常用滤波器**：
- Ram-Lak
- Shepp-Logan
- Hamming
- Butterworth


### 噪声类型与适用滤波器

| 噪声类型 | 推荐滤波器 |
|----------|------------|
| 高斯噪声 | 均值滤波、维纳滤波 |
| 椒盐噪声 | 中值滤波、最大/最小值滤波 |
| 周期噪声 | 陷波滤波器 |

### 复原滤波器对比

| 方法 | 优点 | 缺点 |
|------|------|------|
| 逆滤波 | 简单 | 噪声放大 |
| 维纳滤波 | 考虑噪声 | 需要噪声功率谱 |
| 约束最小二乘 | 稳定 | 需要调参 |
