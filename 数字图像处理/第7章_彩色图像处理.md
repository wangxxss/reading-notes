# 第7章 彩色图像处理

## 学习目标

- 理解不同颜色模型的特点和应用场景
- 掌握颜色模型之间的转换方法
- 理解伪彩色处理和全彩色处理
- 掌握彩色图像增强方法
- 了解彩色图像分割的基本概念

## 7.1 彩色基础

### 7.1.1 可见光谱

**波长范围**：380nm - 780nm

| 颜色 | 波长范围 (nm) |
|------|--------------|
| 紫色 | 380-450 |
| 蓝色 | 450-495 |
| 绿色 | 495-570 |
| 黄色 | 570-590 |
| 橙色 | 590-620 |
| 红色 | 620-750 |

### 7.1.2 颜色的物理属性

| 属性 | 说明 |
|------|------|
| **亮度 (Intensity)** | 光波的振幅，表示光的强弱 |
| **色调 (Hue)** | 光波的波长，决定颜色种类 |
| **饱和度 (Saturation)** | 颜色的纯度，接近单色光的程度 |

### 7.1.3 彩色图像的表示

**灰度图像**：f(x,y) → 单通道强度值

**彩色图像**：f(x,y) → 多通道向量
```
f(x,y) = [f_red(x,y), f_green(x,y), f_blue(x,y)]
```

## 7.2 颜色模型

### 7.2.1 RGB模型

**原理**：加色模型，用三基色组合产生各种颜色

**坐标表示**：
```
颜色 C = rR + gG + bB
其中 r + g + b = 1
```

**颜色立方体**：
```
      (0,1,1)
         /B
        /  (0,0,1)
       /   /|
      /  / G| (0,1,0)
(1,0,1)/  /  |/
 ──────/──/───(1,0,0)
 |    /  /  /
 |   /  /  /R
 |  /  /  /
 | /  /  /
 |/  /  /
(1,1,1)   (1,1,0)
   /  (0,1,0)
  / G (0,0,0) R
 /
B(0,0,1)
```

**特点**：
- 用于显示器、摄像机
- 硬件相关
- 三通道，各256级 (0-255)

### 7.2.2 CMY和CMYK模型

**CMY模型**（减色模型）：
```
C = 1 - R
M = 1 - G
Y = 1 - B
```

**应用**：
- 彩色打印
- 油墨混合

**CMYK模型**：
- 增加黑色 (K) 分量
- 节省墨水
- 更深的黑色

### 7.2.3 HSI模型

**更符合人眼感知**：

| 分量 | 说明 | 范围 |
|------|------|------|
| **H (Hue)** | 色调 | 0°-360° |
| **S (Saturation)** | 饱和度 | 0-1 |
| **I (Intensity)** | 强度 | 0-1 |

**特点**：
- 强度 I 与颜色信息独立
- 更适合图像处理
- 便于分离颜色和亮度信息

### 7.2.4 YIQ和YUV模型

**用于电视广播**：

**YIQ (NTSC制式)**：
- Y = 亮度分量
- I = 同相色彩分量
- Q = 正交色彩分量

**YUV (PAL制式)**：
- Y = 亮度分量
- U, V = 色度分量

**转换公式**：
```
Y = 0.299R + 0.587G + 0.114B
I = 0.596R - 0.275G - 0.321B
Q = 0.212R - 0.523G + 0.311B
```

### 7.2.5 XYZ和CIE模型

**CIE 1931色彩空间**：
- 独立于设备
- 国际标准
- 用于色彩管理

## 7.3 颜色模型转换

### 7.3.1 RGB ↔ HSI

**RGB → HSI**：
```
I = (R + G + B) / 3

S = 1 - (3/(R+G+B)) × min(R,G,B)

H = cos⁻¹{ [(R-G) + (R-B)] / [2√((R-G)² + (R-B)(G-B))] }
    if B > G: H = 360° - H
```

### 7.3.2 RGB ↔ CMY

**RGB → CMY**：
```
C = 255 - R
M = 255 - G
Y = 255 - B
```

### 7.3.3 RGB ↔ YIQ

**RGB → YIQ**：
```
Y = 0.299R + 0.587G + 0.114B
I = 0.596R - 0.275G - 0.321B
Q = 0.212R - 0.523G + 0.311B
```

## 7.4 伪彩色图像处理

### 7.4.1 基本概念

**伪彩色处理**：将灰度图像映射为彩色图像

**目的**：
- 增强人眼对灰度差异的分辨能力
- 使细节更容易识别
- 用于可视化热成像、卫星图像等

### 7.4.2 强度分层法

**原理**：将灰度级划分为若干区间，每个区间映射到不同颜色

```
灰度级 L = 0,1,2,...,L-1
划分点：l₁,l₂,...,lₙ
每个区间赋予不同颜色
```

### 7.4.3 灰度到彩色的变换

**方法**：
- 分别对R、G、B通道应用不同变换
- 组合产生彩色输出

**常见映射**：
| 灰度范围 | 显示颜色 |
|----------|----------|
| 低 | 蓝色 |
| 中 | 绿色 |
| 高 | 红色 |

## 7.5 全彩色图像处理

### 7.5.1 基本流程

```
彩色图像 → 分离通道 → 各通道处理 → 合并通道 → 输出
```

### 7.5.2 彩色平衡

**原因**：
- 传感器响应不均衡
- 光源色温变化

**方法**：
- 灰度世界假设
- 白平衡校正

### 7.5.3 彩色图像增强

**方法**：
1. **各通道独立处理**：分别对R、G、B进行增强
2. **转换到HSI处理**：在HSI空间处理后转回RGB

**HSI空间处理优势**：
- 亮度通道独立处理不影响色调
- 更符合人类视觉特性

---

## 7.6 彩色图像平滑

### 7.6.1 概念

**邻域平均**：对每个像素的邻域内所有像素进行平均

### 7.6.2 各通道分别平滑

```
RGB空间：
Ĉ(x,y) = (1/K) Σ Σ C(s,t)
          s∈S  t∈S

分别对 R, G, B 三个通道进行平滑
```

### 7.6.3 向量直接处理

```
Ĉ(x,y) = (1/K) Σ Σ C(s,t)
          s∈S  t∈S

在彩色向量空间直接计算
```

**注意**：两种方法在理论上不等价

## 7.7 彩色图像锐化

### 7.7.1 方法

**拉普拉斯算子**：
```
∇²[C(x,y)] = [∇²R, ∇²G, ∇²B]
```

**梯度法**：
```
梯度分量使用彩色图像计算
|G| = √(Gx² + Gy²)
```

### 7.7.2 实现

**步骤**：
1. 计算各通道的梯度
2. 组合梯度向量
3. 应用锐化增强

## 7.8 彩色图像分割

### 7.8.1 HSI空间分割

**基于色调H分割**：
```
在HSI空间中对H分量进行阈值分割
适合颜色差异明显的场景
```

**基于饱和度S分割**：
```
分割高饱和度区域
提取鲜艳颜色区域
```

### 7.8.2 RGB空间分割

**使用欧氏距离**：
```
|C - C₀| = √[(R-R₀)² + (G-G₀)² + (B-B₀)²]
```

**使用马氏距离**（考虑相关性）：
```
d = √[(C-μ)ᵀ Σ⁻¹ (C-μ)]
```

### 7.8.3 边缘检测

**方法**：
1. 各通道分别边缘检测
2. 组合边缘结果
3. 使用向量梯度方法

## 7.9 彩色图像的噪声

### 7.9.1 噪声类型

| 噪声 | 特点 |
|------|------|
| 单通道噪声 | 某一通道特定噪声 |
| 多通道噪声 | 影响多个通道 |
| 相关噪声 | 通道间相关 |

### 7.9.2 处理方法

**各通道独立滤波**：
- 分别对R、G、B滤波
- 中值滤波对椒盐噪声有效

**向量滤波**：
- 在向量空间直接处理
- 保持通道间关系

## 本章小结

### 颜色模型对比

| 模型 | 类型 | 应用 |
|------|------|------|
| RGB | 加色 | 显示器、摄像机 |
| CMY/K | 减色 | 打印 |
| HSI | 感知型 | 图像处理 |
| YIQ/YUV | 广播型 | 电视 |
| XYZ | CIE标准 | 色彩管理 |

### 彩色图像处理方法

| 操作 | 方法 |
|------|------|
| 增强 | HSI空间处理 |
| 平滑 | 邻域平均 |
| 锐化 | 拉普拉斯/梯度 |
| 分割 | H阈值、距离度量 |

### 关键点

1. **HSI模型更符合人眼感知**，适合图像处理
2. **伪彩色**用于增强灰度图像的可视性
3. **彩色平衡**校准传感器响应差异
4. **向量方法**和**分量方法**处理结果不同
