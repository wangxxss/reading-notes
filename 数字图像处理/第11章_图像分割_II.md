## 11.1 主动轮廓模型概述

### 11.1.1 基本概念

**主动轮廓 (Active Contour)**：通过定义一条初始轮廓曲线，并在内外力的共同驱动下，使其不断变形、移动，最终与目标物体的边界精确重合。

### 11.1.2 发展历史

| 年份 | 方法 | 特点 |
|------|------|------|
| 1988 | Snakes (Kass) | 经典主动轮廓 |
| 1989 | balloon | 改进的Snakes |
| 1993 | T-Snakes | 张力样条 |
| 1997 | GVFs | 梯度向量流 |
| 1997 | Level Sets | 水平集方法 |

### 11.1.3 主动轮廓的分类

## 11.2 Snakes模型

### 11.2.1 基本原理

**能量函数**：
```
E = E_internal + E_external + E_constraint
```

**轮廓表示**：
```
v(s) = [x(s), y(s)], s ∈ [0, 1]
```

### 11.2.2 内部能量

**弹性能量**（一阶导数）：
```
E_internal = α(s) |∂v/∂s|²
```

**弯曲能量**（二阶导数）：
```
E_bending = β(s) |∂²v/∂s²|²
```

**作用**：
- α 控制轮廓弹性（拉伸）
- β 控制轮廓刚性（平滑）

### 11.2.3 外部能量

**图像能量**：
```
E_image = -|∇f|² (边缘驱动)
或
E_image = -(f - f_edge)² (区域驱动)
```

**外部力场**：
```
F_ext = -∇E_external
```

### 11.2.4 能量最小化

**变分法求解**：
```
∂E/∂v - ∂/∂s(∂E/∂v_s) + ∂²/∂s²(∂E/∂v_ss) = 0
```

**迭代求解**：
```
vⁿ⁺¹ = vⁿ + γ(F_ext + F_internal)
```

### 11.2.5 Snakes的优缺点

**优点**：
- 边界光滑连续
- 可融合先验知识
- 理论基础好

**缺点**：
- 对初始位置敏感
- 难以收敛到深凹区域
- 拓扑适应性差

## 11.3 改进的Snakes模型

### 11.3.1 Balloon模型

**改进**：增加法向膨胀力

```
F = k₁n(s) - ∇p(v(s))
```

**作用**：
- 推动轮廓向外/向内膨胀
- 帮助穿越伪边缘

### 11.3.2 GVF (Gradient Vector Flow)

**原理**：扩散梯度场

**GVF场定义**：
```
∂u/∂t = ∇²u - (u - ∂f/∂x)(∂²f/∂x² + ∂²f/∂y²)
∂v/∂t = ∇²v - (v - ∂f/∂y)(∂²f/∂x² + ∂²f/∂y²)
```

**优点**：
- 扩大搜索范围
- 收敛到深凹区域
- 对初始位置不敏感

## 11.4 Level Set方法

### 11.4.1 基本概念

**隐式表示**：
```
φ(x,y,t) = 0 表示轮廓
```

**符号距离函数 (SDF)**：
```
φ(x,y) = { +d, if outside
          {  0, if on contour
          { -d, if inside
d = 到最近轮廓点的距离
```

### 11.4.2 水平集演化方程

**基本公式**：
```
∂φ/∂t + F|∇φ| = 0
```

**速度场F**：
```
F = F_advection + F_curvature + F_reaction
```

### 11.4.3 曲率驱动演化

```
∂φ/∂t = κ|∇φ|
其中 κ = div(∇φ/|∇φ|) = 曲率
```

**作用**：
- 平滑轮廓
- 消除小突起
- 类似于曲线缩短流

### 11.4.4 Level Set的优缺点

**优点**：
- 拓扑自适应（自动分裂/合并）
- 数值稳定
- 易于扩展

**缺点**：
- 计算量大
- 符号距离函数需要初始化
- 窄带方法加速计算

## 11.5 图割 (Graph Cuts) 方法

### 11.5.1 基本概念

**图割**：将图像分割问题转化为图的最小割问题

**图结构**：
```
┌─────────────────────────────────────┐
│         图 (Graph)                  │
│                                     │
│   源点 (Source) ──┬──→ 像素节点     │
│                   │                │
│   汇点 (Sink)  ──┴──→ 像素节点     │
│                                     │
└─────────────────────────────────────┘
```

### 11.5.2 图割能量函数

```
E(f) = λ × R(f) + B(f)

其中：
  R(f) = Σ Dₚ(fₚ)  区域项
  B(f) = Σ B_{p,q} × |fₚ - f_q|  边界项
```

### 11.5.3 Boykov算法

**步骤**：
1. 构建图结构
2. 计算最小割
3. 得到分割结果

**应用**：
- 交互式分割
- 立体匹配
- 图像修复

## 11.6 超级像素 (Superpixels)

### 11.6.1 基本概念

**超级像素**：具有相似颜色/纹理特征的相邻像素组成的区域

**优点**：
- 减少后续处理复杂度
- 保留边界
- 计算效率高

### 11.6.2 SLIC算法

**SLIC** = Simple Linear Iterative Clustering

**步骤**：
```
1. 初始化种子点(均匀分布)
2. 在邻域内搜索相似像素
3. 更新聚类中心
4. 迭代直到收敛
```

**距离度量**：
```
d = √(d²_color + m²×d²_space)
```

### 11.6.3 MSER

**MSER** = Maximally Stable Extremal Regions

**原理**：检测灰度变化平稳的区域

**应用**：
- 特征检测
- 宽基线匹配
- 场景识别

## 11.7 k-means聚类

### 11.7.1 基本原理

**目标函数**：
```
J = Σ Σ ||xᵢ - μⱼ||²
    j   i∈Cⱼ
```

### 11.7.2 算法步骤

```
1. 初始化K个聚类中心
2. 分配每个样本到最近的聚类中心
3. 更新聚类中心为该类样本的均值
4. 重复步骤2-3直到收敛
```

### 11.7.3 在图像分割中的应用

**特征**：颜色、纹理、位置

**应用**：
- 颜色分割
- 纹理分割
- 运动分割

## 11.8 分割方法比较

| 方法 | 优点 | 缺点 | 适用场景 |
|------|------|------|----------|
| Snakes | 边界光滑 | 拓扑不变性差 | 医学图像 |
| Level Sets | 拓扑自适应 | 计算慢 | 复杂形状 |
| Graph Cuts | 全局最优 | 需要交互 | 交互分割 |
| SLIC | 效率高 | 边界精度 | 预处理 |
| k-means | 简单 | 对初始敏感 | 颜色分割 |
