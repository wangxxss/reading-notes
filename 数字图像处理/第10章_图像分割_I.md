## 10.1 基本概念

### 10.1.1 图像分割定义

**定义**：将图像划分为若干具有独特特性的区域，并把感兴趣的目标分离出来的过程。

### 10.1.2 分割的数学描述

设 R 代表整个图像区域，分割是将 R 划分为 n 个子区域 R₁, R₂, ..., Rₙ 的过程，满足：

```
(a) Σ Rᵢ = R
(b) Rᵢ 是连通的
(c) Rᵢ ∩ Rⱼ = ∅, i ≠ j
(d) P(Rᵢ) = TRUE (区域内像素满足某属性)
(e) P(Rᵢ ∪ Rⱼ) = FALSE, i ≠ j (相邻区域属性不同)
```

### 10.1.3 分割方法的分类

| 类别 | 方法 |
|------|------|
| **基于边缘** | 边缘检测、边界追踪 |
| **基于区域** | 阈值分割、区域生长、区域分裂合并 |
| **组合方法** | 分水岭、混合方法 |

## 10.2 边缘检测

### 10.2.1 基本概念

**边缘**：图像中灰度发生急剧变化的像素集合

**边缘类型**：
```
         阶梯边缘         斜坡边缘         线边缘
         ████████         ████            ██
                          ████████
       ████               ████            ██
       ████                            ████
```

### 10.2.2 边缘检测的基本步骤

```
图像 → 平滑去噪 → 梯度计算 → 阈值处理 → 边缘定位
```

### 10.2.3 一阶微分边缘检测

**梯度定义**：
```
∇f = [Gₓ, Gᵧ]ᵀ = [∂f/∂x, ∂f/∂y]ᵀ
```

**梯度幅值**：
```
|∇f| = √(Gₓ² + Gᵧ²)
```

**梯度方向**：
```
φ = tan⁻¹(Gᵧ / Gₓ)
```

### 10.2.4 Roberts交叉梯度算子

**模板**：
```
Gₓ: [ 1  0 ]      Gᵧ: [ 0  1 ]
    [ 0 -1 ]          [-1  0]
```

**特点**：
- 简单
- 对角线响应强
- 噪声敏感

### 10.2.5 Prewitt算子

**模板**：
```
Gₓ: [-1  0  1]      Gᵧ: [-1 -1 -1]
    [-1  0  1]          [ 0  0  0]
    [-1  0  1]          [ 1  1  1]
```

**特点**：
- 平滑噪声
- 计算简单

### 10.2.6 Sobel算子

**模板**：
```
Gₓ: [-1  0  1]      Gᵧ: [-1 -2 -1]
    [-2  0  2]          [ 0  0  0]
    [-1  0  1]          [ 1  2  1]
```

**特点**：
- 噪声抑制更好
- 更常用

## 10.3 二阶微分边缘检测

### 10.3.1 拉普拉斯算子

**定义**：
```
∇²f = ∂²f/∂x² + ∂²f/∂y²
```

**离散形式**：
```
∇²f = f(x+1,y) + f(x-1,y) + f(x,y+1) + f(x,y-1) - 4f(x,y)
```

**模板 (4-邻域)**：
```
[ 0 -1  0
 -1  4 -1
  0 -1  0]
```

**8-邻域模板**：
```
[-1 -1 -1
 -1  8 -1
 -1 -1 -1]
```

### 10.3.2 拉普拉斯算子的特点

**优点**：
- 边缘定位准确
- 对所有方向响应均匀

**缺点**：
- 噪声放大
- 双边缘效应
- 不能检测边缘方向

### 10.3.3 高斯拉普拉斯 (LoG)

**定义**：
```
LoG(x,y) = ∇²[G(x,y) * f(x,y)] = G(x,y) * ∇²f(x,y)
```

**高斯函数**：
```
G(x,y) = (1/2πσ²) × e^(-(x²+y²)/2σ²)
```

**特点**：
- 先平滑后锐化
- 消除噪声
- 边缘检测+定位

**零交叉检测**：
```
找到 LoG 输出穿过零点位置
```

## 10.4 Canny边缘检测器

### 10.4.1 算法步骤

```
┌──────────────────────────────────────────────────────────┐
│ 1. 高斯平滑                                              │
│    使用高斯滤波器平滑图像，减少噪声                      │
└──────────────────────────────────────────────────────────┘
                         ↓
┌──────────────────────────────────────────────────────────┐
│ 2. 梯度计算                                              │
│    计算梯度幅值和方向                                    │
│    |∇f| = √(Gₓ² + Gᵧ²), φ = tan⁻¹(Gᵧ/Gₓ)               │
└──────────────────────────────────────────────────────────┘
                         ↓
┌──────────────────────────────────────────────────────────┐
│ 3. 非极大值抑制 (NMS)                                    │
│    细化边缘，只保留局部最大值点                          │
└──────────────────────────────────────────────────────────┘
                         ↓
┌──────────────────────────────────────────────────────────┐
│ 4. 滞后阈值处理                                          │
│    使用双阈值 (T₁, T₂)                                  │
│    - 强边缘 > T₂: 保留                                  │
│    - 弱边缘 T₁-T₂: 连通强边缘则保留                     │
│    - 噪声 < T₁: 丢弃                                    │
└──────────────────────────────────────────────────────────┘
                         ↓
┌──────────────────────────────────────────────────────────┐
│ 5. 边缘连接                                              │
│    消除假边缘，形成连续边界                             │
└──────────────────────────────────────────────────────────┘
```

### 10.4.2 Canny算子的优点

| 优点 | 说明 |
|------|------|
| 好的检测 | 不漏检真实边缘 |
| 好的定位 | 边缘位置准确 |
| 最小响应 | 每个边缘只有一个响应 |

### 10.4.3 边缘检测算子比较

| 算子 | 优点 | 缺点 | 适用场景 |
|------|------|------|----------|
| Roberts | 边缘定位准 | 噪声敏感 | 低噪声 |
| Prewitt | 简单 | 边缘粗 | 一般 |
| Sobel | 噪声抑制 | 边缘粗 | 一般 |
| LoG | 定位准 | 计算复杂 | 多尺度 |
| Canny | 最优 | 计算复杂 | 通用 |

## 10.5 阈值分割

### 10.5.1 基本概念

**阈值分割**：根据灰度值将图像分为目标和背景

**基本公式**：
```
g(x,y) = 1,  if f(x,y) > T
       = 0,  if f(x,y) ≤ T
```

### 10.5.2 全局阈值

**全局阈值T**：适用于整幅图像

**Otsu方法**：

**原理**：最大化类间方差

```
σ²_B = w₁w₂(μ₁ - μ₂)²
```

**步骤**：
1. 计算图像直方图
2. 遍历所有可能的阈值T
3. 计算类间方差
4. 选择使σ²_B最大的T

### 10.5.3 自适应阈值

**问题**：光照不均匀时，全局阈值效果差

**解决方法**：局部阈值

```
T(x,y) = mean(I(x,y)) - C
或
T(x,y) = 加权均值 - C
```

### 10.5.4 多阈值分割

**公式**：
```
g(x,y) = k,  if Tₖ < f(x,y) ≤ Tₖ₊₁
```

**应用**：
- 多目标分割
- 灰度图像分区

## 10.6 区域生长

### 10.6.1 基本概念

**区域生长**：将具有相似性质的像素聚合为同一区域

### 10.6.2 算法步骤

```
1. 选择种子点 (Seed)
2. 确定生长准则 (Similarity)
3. 确定停止条件 (Stop criteria)
4. 迭代生长
5. 输出结果
```

### 10.6.3 生长准则

| 准则 | 公式 |
|------|------|
| 灰度差 | |f(x,y) - f(xᵢ,yᵢ)| < T |
| 灰度均值 | |f(x,y) - μ| < T |
| 纹理 | 纹理模式相同 |

### 10.6.4 区域分裂与合并

**分裂**：将不均匀区域分成四象限

**合并**：合并相似的相邻区域

**分裂合并算法**：
```
1. 分裂: Q(A) = FALSE → 分裂A为4块
2. 合并: Q(A∪B) = TRUE → 合并A和B
3. 迭代直到没有变化
```

## 10.7 分水岭算法

### 10.7.1 基本概念

**分水岭**：基于地形学的分割方法

**类比**：
- 图像灰度 → 地形高度
- 集水盆 → 目标区域
- 分水岭 → 区域边界

### 10.7.2 算法步骤

```
┌──────────────────────────────────────┐
│ 1. 将图像视为地形表面                  │
└──────────────────────────────────────┘
                    ↓
┌──────────────────────────────────────┐
│ 2. 从低洼点(种子)开始注水              │
└──────────────────────────────────────┘
                    ↓
┌──────────────────────────────────────┐
│ 3. 水位上升，相邻低洼合并              │
└──────────────────────────────────────┘
                    ↓
┌──────────────────────────────────────┐
│ 4. 不同区域汇水时形成分水岭线          │
└──────────────────────────────────────┘
```

### 10.7.3 标记分水岭

**问题**：噪声导致过分割

**解决方法**：使用标记

```
1. 标记前景物体
2. 标记背景区域
3. 应用分水岭变换
```

### 10.7.4 分水岭算法的应用

| 应用 | 说明 |
|------|------|
| 细胞图像分割 | 分离重叠细胞 |
| 颗粒分析 | 分割重叠颗粒 |
| 医学图像 | 组织分割 |

## 10.8 分割算法评价

### 10.8.1 评价准则

| 准则 | 说明 |
|------|------|
| 区域一致性 | 区域内灰度/纹理相似 |
| 区域形状 | 边界光滑 |
| 计时性能 | 计算复杂度 |

### 10.8.2 常用评价方法

**有参考分割评价**：
```
- 概率Rand指数
- Variation of Information
- F-measure
```

**无参考分割评价**：
```
- 区域间对比度
- 内部一致性
```
