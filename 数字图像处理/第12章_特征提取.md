# 第12章 特征提取

## 学习目标

- 理解特征提取的基本概念
- 掌握边界特征提取方法
- 理解区域特征描述
- 掌握纹理特征分析方法
- 了解矩特征和形状描述

---

## 12.1 基本概念

### 12.1.1 特征的定义

**图像特征**：用于描述图像或图像中物体属性的定量测量

### 12.1.2 特征的分类

| 类别 | 示例 |
|------|------|
| **边界特征** | 边缘、轮廓、形状 |
| **区域特征** | 面积、周长、质心 |
| **纹理特征** | 粗糙度、方向性、对比度 |
| **统计特征** | 均值、方差、直方图 |

### 12.1.3 特征的要求

| 要求 | 说明 |
|------|------|
| **可区分性** | 不同类别的特征值差异大 |
| **可靠性** | 同一类别特征值相近 |
| **独立性** | 特征间不相关 |
| **数量适当** | 避免维度灾难 |

## 12.2 边界特征

### 12.2.1 边界追踪

**算法步骤**：
```
1. 从左到右扫描找到起始边界点
2. 从8-邻域中寻找下一边界点
3. 重复直到回到起始点
```

**链码 (Chain Code)**：
```
方向: 0 1 2 3 4 5 6 7
      ↑ ↗ → ↘ ↓ ↙ ← ↖
```

** Freeman链码**：用方向序列表示边界

### 12.2.2 边界长度

**计算方法**：
```
对于水平/垂直步长: 加1
对于对角步长: 加√2
```

### 12.2.3 边界直径

**定义**：边界上两点间的最大欧氏距离

**直径点**：取得直径的两点

**应用**：形状描述

## 12.3 形状描述

### 12.3.1 边界形状描述

**简单形状参数**：

| 参数 | 公式 |
|------|------|
| 紧凑度 | P² / A |
| 偏心度 | 长轴 / 短轴 |
| 矩形度 | A / A_rect |
| 圆形度 | 4πA / P² |

### 12.3.2 傅里叶描述子

**基本原理**：
1. 将边界表示为复数序列
2. 进行傅里叶变换
3. 使用变换系数作为描述子

**复数表示**：
```
s(k) = x(k) + jy(k), k = 0, 1, ..., N-1
```

**傅里叶描述子**：
```
S(n) = Σ s(k) e^(-j2πnk/N)
```

**特点**：
- 旋转、平移、缩放不变性
- 紧凑表示
- 重建原始边界

### 12.3.3 矩描述子

**几何矩**：
```
m_{pq} = Σ Σ x^p y^q f(x,y)
         x   y
```

**中心矩**：
```
μ_{pq} = Σ Σ (x-x̄)^p (y-ȳ)^q f(x,y)
         x   y

其中 x̄ = m₁₀/m₀₀, ȳ = m₀₁/m₀₀
```

**归一化中心矩**：
```
η_{pq} = μ_{pq} / μ₀₀^((p+q)/2 + 1)
```

**Hu矩**（7个不变矩）：
```
φ₁ = η₂₀ + η₀₂
φ₂ = (η₂₀ - η₀₂)² + 4η₁₁²
φ₃ = (η₃₀ - 3η₁₂)² + (3η₂₁ - η₀₃)²
...
```

**特点**：平移、旋转、缩放不变

## 12.4 区域特征

### 12.4.1 面积

**计算方法**：
```
A = Σ Σ f(x,y)
    x   y
```

**或统计前景像素个数**

### 12.4.2 周长

**计算方法**：
```
P = 边界像素数 × 单位长度
```

### 12.4.3 质心

**定义**：
```
x̄ = (1/A) Σ Σ x × f(x,y)
ȳ = (1/A) Σ Σ y × f(x,y)
```

### 12.4.4 区域描述参数

| 参数 | 公式 | 说明 |
|------|------|------|
| 致密度 | 4πA/P² | 接近1为圆形 |
| 矩形度 | A/A_min | 充满最小外接矩形的程度 |
| 伸长度 | min(L,W)/max(L,W) | 长宽比 |

## 12.5 纹理特征

### 12.5.1 纹理的基本概念

**纹理**：图像像素灰度级的空间分布规律

**类型**：
| 类型 | 示例 |
|------|------|
| 人工纹理 | 砖墙、织物 |
| 自然纹理 | 树木、云彩 |
| 统计纹理 | 沙地、草地 |

### 12.5.2 纹理分析方法

```
┌─────────────────────────────────┐
│        纹理分析方法              │
├─────────────┬───────────────────┤
│ 统计方法    │ 结构方法          │
├─────────────┼───────────────────┤
│ 共生矩阵    │ 纹理梯度          │
│ 纹理能量    │ 边缘密度          │
│ LBP         │ 形态学            │
└─────────────┴───────────────────┘
```

### 12.5.3 灰度共生矩阵 (GLCM)

**定义**：像素对灰度值组合的统计

**共生矩阵元素**：
```
P(i,j) = {(x,y)和(x+Δx,y+Δy)处灰度为(i,j)的次数}
         / 总共生对数
```

**特征参数**：

| 特征 | 公式 | 说明 |
|------|------|------|
| 对比度 | Σ (i-j)² P(i,j) | 纹理的清晰程度 |
| 均匀性 | Σ P(i,j)² | 纹理的规则程度 |
| 能量 | Σ P(i,j)² | 灰度分布均匀性 |
| 相关性 | ΣᵢΣⱼ ijP(i,j) - μₓμᵧ / σₓσᵧ | 灰度线性关系 |

### 12.5.4 局部二值模式 (LBP)

**基本原理**：
```
1. 比较中心像素与8邻域像素
2. 大于中心: 1, 否则: 0
3. 形成二进制模式
```

**统一LBP**：
```
统计模式出现的次数
用于纹理分类
```

**特点**：
- 计算简单
- 旋转不变性
- 对比度信息

### 12.5.5 纹理频谱

**傅里叶频谱**：
- 低频能量：均匀纹理
- 高频能量：精细纹理
- 方向性：频谱分布方向

## 12.6 霍夫变换

### 12.6.1 直线检测

**参数空间**：
```
ρ = x cosθ + y sinθ
ρ: 原点到直线的距离
θ: 直线法向角度
```

**Hough变换**：
```
图像空间 → 参数空间
点 → 正弦曲线
共线点 → 相交的正弦曲线
```

### 12.6.2 圆检测

**参数方程**：
```
(x - a)² + (y - b)² = r²
```

**参数空间**：三维 (a, b, r)

### 12.6.3 广义Hough变换

**用途**：检测任意形状

**原理**：
1. 建立R-table：边界点到参考点的距离和方向
2. 投票累加
3. 找到峰值

## 12.7 SIFT特征

### 12.7.1 基本概念

**SIFT** = Scale-Invariant Feature Transform

**特点**：
- 尺度不变性
- 旋转不变性
- 光照不变性

### 12.7.2 算法步骤

```
1. 尺度空间极值检测 (DoG)
2. 关键点定位
3. 方向分配
4. 关键点描述子生成
```

### 12.7.3 应用

| 应用 | 说明 |
|------|------|
| 目标识别 | 匹配关键点 |
| 图像配准 | 几何变换估计 |
| 3D重建 | 多视角匹配 |
| 图像拼接 | 特征对应 |

## 12.8 特征选择与降维

### 12.8.1 特征选择

**目的**：去除冗余特征

**方法**：
- 穷举搜索
- 序列前向/后向选择
- 遗传算法

### 12.8.2 PCA降维

**主成分分析**：
```
1. 计算协方差矩阵
2. 求特征值和特征向量
3. 选择主要成分
```

**应用**：
- 特征压缩
- 数据可视化
- 人脸识别 (Eigenfaces)

## 本章小结

### 特征类型总结

| 类型 | 特征 | 应用 |
|------|------|------|
| 边界 | 链码、傅里叶描述子 | 形状分析 |
| 区域 | 面积、周长、质心 | 目标描述 |
| 纹理 | GLCM、LBP、频谱 | 表面分析 |
| 关键点 | SIFT、HOG | 匹配识别 |

### 纹理分析方法对比

| 方法 | 优点 | 缺点 |
|------|------|------|
| GLCM | 物理意义明确 | 计算量大 |
| LBP | 简单高效 | 尺度不变性差 |
| 频谱 | 方向性分析好 | 需要周期性纹理 |

### Hu矩的不变性

1. 平移不变性
2. 旋转不变性
3. 缩放不变性
