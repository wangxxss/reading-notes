# 第9章 形态学图像处理

## 学习目标

- 理解数学形态学的基本概念
- 掌握腐蚀和膨胀的基本操作
- 理解开操作和闭操作的作用
- 掌握击中击不中变换
- 了解形态学在边界提取、区域填充等中的应用

---

## 9.1 形态学基本概念

### 9.1.1 数学形态学定义

**数学形态学 (Mathematical Morphology)**：基于集合论的语言，用于分析和处理图像中物体的几何形状和结构。

### 9.1.2 基本思想

**核心思想**：使用一个"探针"（结构元素）在图像中移动，研究图像中物体的形状和结构。

**结构元素 (Structuring Element, SE)**：
```
┌────────────────────────┐
│  示例 3×3 结构元素:     │
│                         │
│    [0 1 0]              │
│    [1 1 1]              │
│    [0 1 0]              │
│                         │
│  原点通常在中心          │
└────────────────────────┘
```

**结构元素形状**：
- 方形
- 圆形
- 十字形
- 自定义形状

## 9.2 集合论基础

### 9.2.1 集合运算

| 运算 | 符号 | 定义 |
|------|------|------|
| 交集 | A ∩ B | 同时属于A和B的元素 |
| 并集 | A ∪ B | 属于A或B的元素 |
| 差集 | A - B | 属于A不属于B的元素 |
| 补集 | Aᶜ | 不属于A的元素 |
| 对称差 | A ⊕ B | (A-B) ∪ (B-A) |

### 9.2.2 反射和平移

**反射**：
```
B̂ = {w | w = -b, b ∈ B}
```

**平移**：
```
(B)ᵧ = {c | c = b + y, b ∈ B}
```

## 9.3 二值图像腐蚀和膨胀

### 9.3.1 腐蚀 (Erosion)

**定义**：
```
A ⊖ B = {z | (B)ᵤ ⊆ A}
```

**几何解释**：当结构元素B的原点平移到z时，B完全包含在A中。

**效果**：
- 缩小物体
- 断开窄的连接
- 去除小的突出物
- 去除噪声（细小亮点）

**示例**：
```
原图像:          腐蚀后:
■■■■■■■■        ■■■■■
■■■■■■■■   →    ■■■■■
■■■■■■■■        ■■■■■
(■表示前景)
```

### 9.3.2 膨胀 (Dilation)

**定义**：
```
A ⊕ B = {z | (B̂)ᵤ ∩ A ≠ ∅}
```

**几何解释**：当结构元素B的反射原点平移到z时，B与A有交集。

**效果**：
- 扩大物体
- 连接临近的物体
- 填充小的空洞
- 修复断裂

**示例**：
```
原图像:          膨胀后:
■■■■■          ■■■■■■■
■■■■■    →     ■■■■■■■
■■■■■          ■■■■■■■
```

### 9.3.3 腐蚀与膨胀的对偶性

```
(A ⊖ B)ᶜ = Aᶜ ⊕ B̂
(A ⊕ B)ᶜ = Aᶜ ⊖ B̂
```

## 9.4 开操作与闭操作

### 9.4.1 开操作 (Opening)

**定义**：
```
A ○ B = (A ⊖ B) ⊕ B
```

**步骤**：先腐蚀后膨胀

**效果**：
- 平滑物体轮廓
- 断开窄的狭颈
- 消除细的突出物
- **总体上使物体变小**

**几何解释**：
```
A ○ B = B关于A的"内切圆"的最大半径的轨迹
```

### 9.4.2 闭操作 (Closing)

**定义**：
```
A ● B = (A ⊕ B) ⊖ B
```

**步骤**：先膨胀后腐蚀

**效果**：
- 平滑物体轮廓
- 弥合窄的间断
- 填充小的孔洞
- 填补轮廓线中的断裂
- **总体上使物体变大**

### 9.4.3 开闭操作的对偶性

```
(A ○ B)ᶜ = Aᶜ ● B̂
(A ● B)ᶜ = Aᶜ ○ B̂
```

### 9.4.4 应用选择

| 操作 | 适用场景 |
|------|----------|
| 开操作 | 去除噪声、断开连接 |
| 闭操作 | 填充空洞、连接断裂 |

## 9.5 击中击不中变换 (Hit-or-Miss Transform)

### 9.5.1 基本概念

**定义**：
```
A ⊛ B = (A ⊖ B₁) ∩ (Aᶜ ⊖ B₂)
```

其中 B = (B₁, B₂)，B₁为前景部分，B₂为背景部分

### 9.5.2 几何解释

**击中**：结构元素完全匹配图像的前景部分

**击不中**：结构元素完全匹配图像的背景部分

### 9.5.3 应用

- **形状识别**：检测特定形状
- **细化**：骨架提取
- **修剪**：去除毛刺

---

## 9.6 形态学应用

### 9.6.1 边界提取 (Boundary Extraction)

**公式**：
```
β(A) = A - (A ⊖ B)
```

**步骤**：
1. 用结构元素B腐蚀图像A
2. 从原图中减去腐蚀结果

### 9.6.2 区域填充 (Region Filling)

**公式**：
```
Xₖ = (Xₖ₋₁ ⊕ B) ∩ Aᶜ, k = 1, 2, 3, ...
X₀ = p (种子点)
```

**应用**：
- 填充孔洞
- 连通分量标记

### 9.6.3 连通分量提取 (Extraction of Connected Components)

**公式**：
```
Xₖ = (Xₖ₋₁ ⊕ B) ∩ A, k = 1, 2, 3, ...
X₀ = 种子点
```

**应用**：
- 对象计数
- 细胞分析
- 缺陷检测

### 9.6.4 凸壳 (Convex Hull)

**定义**：包含物体的最小凸多边形

**迭代公式**：
```
Hᵢ = (Hᵢ₋₁ ⊛ Bᵢ) ∪ Hᵢ₋₁
H₀ = A
```

### 9.6.5 细化 (Thinning)

**公式**：
```
A ⊙ B = A - (A ⊛ B)
     = A ∩ (A ⊛ B)ᶜ
```

**效果**：提取物体骨架，保持连通性

### 9.6.6 粗化 (Thickening)

**公式**：
```
A ⊙ B = A ∪ (A ⊛ B)
```

**效果**：使物体变粗

### 9.6.7 骨架 (Skeleton / Medial Axis Transform)

**公式**：
```
S(A) = ∪ Sₖ(A)
Sₖ(A) = (A ⊖ kB) - ((A ⊖ kB) ○ B)
```

**特点**：
- 保持拓扑结构
- 用于形状分析
- 用于字符识别

## 9.7 灰度形态学

### 9.7.1 灰度腐蚀

```
[f ⊖ b](x,y) = min { f(x+s, y+t) - b(s,t) }
```

### 9.7.2 灰度膨胀

```
[f ⊕ b](x,y) = max { f(x-s, y-t) + b(s,t) }
```

### 9.7.3 灰度开闭操作

**灰度开操作**：
```
f ○ b = (f ⊖ b) ⊕ b
```

**灰度闭操作**：
```
f ● b = (f ⊕ b) ⊖ b
```

### 9.7.4 灰度形态学应用

| 应用 | 说明 |
|------|------|
| 颗粒分析 | 分离重叠颗粒 |
| 表面纹理 | 平滑或增强纹理 |
| 图像修复 | 填补划痕 |

## 9.8形态学滤波器

### 9.8.1 基本滤波器

| 滤波器 | 定义 |
|--------|------|
| 形态学开 | Ĉ = (f ⊖ b) ⊕ b |
| 形态学闭 | Ĉ = (f ⊕ b) ⊖ b |
| 交替滤波器 | 开-闭、闭-开组合 |

### 9.8.2 组合滤波器

**开-闭滤波器**：
```
OC = (f ○ b) ● b
```

**闭-开滤波器**：
```
CO = (f ● b) ○ b
```

## 本章小结

### 基本操作对比

| 操作 | 公式 | 效果 |
|------|------|------|
| 腐蚀 | A ⊖ B | 缩小物体，去除噪声 |
| 膨胀 | A ⊕ B | 扩大物体，填充空洞 |
| 开操作 | A ○ B | 平滑，断开狭颈 |
| 闭操作 | A ● B | 平滑，弥合断裂 |

### 结构元素选择

| 结构元素 | 适用场景 |
|----------|----------|
| 圆形 | 保持边缘平滑 |
| 方形 | 通用形状 |
| 十字形 | 保持线性特征 |

### 形态学应用总结

| 应用 | 方法 |
|------|------|
| 边界提取 | A - (A ⊖ B) |
| 区域填充 | 迭代膨胀+交集 |
| 连通分量 | 迭代膨胀+交集 |
| 骨架 | 反复腐蚀+开操作差 |
| 细化 | 击中击不中变换 |
