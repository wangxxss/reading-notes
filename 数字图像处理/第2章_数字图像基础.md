## 2.1 视觉感知要素

### 2.1.1 人眼的结构

```
┌─────────────────────────────────────┐
│              角膜 (Cornea)           │
│  ┌───────────────────────────────┐  │
│  │           虹膜 (Iris)          │  │
│  │    ┌─────────────────────┐    │  │
│  │    │    瞳孔 (Pupil)      │    │  │
│  │    └─────────────────────┘    │  │
│  │         晶状体 (Lens)         │  │
│  │              ↓                │  │
│  │         视网膜 (Retina)        │  │
│  │    ┌─────────────────────┐    │  │
│  │    │  视杆细胞 (Rods)     │    │  │
│  │    │  视锥细胞 (Cones)    │    │  │
│  │    └─────────────────────┘    │  │
│  └───────────────────────────────┘  │
└─────────────────────────────────────┘
```

### 2.1.2 眼睛中图像的形成

- 物体通过晶状体在视网膜上形成**倒立的实像**
- 晶状体相当于可变焦距的透镜
- 睫状肌调节晶状体厚度以适应不同距离的物体

### 2.1.3 亮度适应与马赫带效应

**亮度适应**：
- 人眼可适应的亮度范围极广（10^(-10) 到 10^5 尼特）
- 通过瞳孔调节和化学适应实现
- Weber-Fechner 定律：ΔI/I = 常数

**马赫带效应**：
- 在灰度级相同的条带中
- 靠近暗带的边缘显得更亮
- 靠近亮带的边缘显得更暗

### 2.1.4 同时对比度

- 同一灰度在不同的背景亮度下看起来不同
- 背景较亮时，物体看起来较暗
- 背景较暗时，物体看起来较亮

## 2.3 图像感知与获取

### 2.3.1 单个传感器获取图像

**优点**：
- 结构简单，成本低
- 可定制特殊波段传感器
- 功耗低

**缺点**：
- 帧率低，难拍运动物体
- 分辨率受限
- 机械扫描易受振动影响

### 2.3.2 条带传感器获取图像

**优点**：
- 高分辨率
- 大视场适应性
- 高感光效率
- 易于定制多波段

**缺点**：
- 依赖运动同步
- 帧率受限
- 系统复杂度高

### 2.3.3 传感器阵列获取图像

**优点**：
- 瞬时成像
- 系统简单
- 高帧率潜力
- 彩色成像成熟

**缺点**：
- 视场限制
- 像素限制
- 大尺寸成本高
- 卷帘快门畸变

## 2.4 图像采样与量化（★★★★★）

### 2.4.1 基本概念

**采样 (Sampling)**：
- 将连续的空间坐标转换为离散值
- 决定图像的**空间分辨率**
- 采样定理：采样频率 ≥ 2 × 最高信号频率

**量化 (Quantization)**：
- 将连续的灰度值映射到有限的整数集合
- 决定图像的**灰度级数**
- 位深度 k 决定灰度级数 L = 2^k

### 2.4.2 数字图像的数学表示

```
数字图像 = M × N × L
其中：
  M = 行数（高度）
  N = 列数（宽度）
  L = 灰度级数（2^k）
```

**示例**：
- 1024 × 1024 的 8-bit 图像
- 存储大小 = 1024 × 1024 × 8 bits = 1 MB

### 2.4.3 采样和量化对图像质量的影响

| 参数 | 值增加 | 值减少 |
|------|--------|--------|
| 采样率 (M, N) | 分辨率↑，细节更清晰 | 块效应、混叠 |
| 量化级数 (L, k) | 平滑过渡，伪轮廓↓ | 伪轮廓、颗粒感↑ |

### 2.4.4 图像分辨率

**空间分辨率**：
- 图像中可分辨的最小细节
- 通常用 DPI (Dots Per Inch) 表示

**灰度分辨率**：
- 相邻灰度级间的可区分程度
- 由位深度决定

## 2.5 像素间的基本关系

### 2.5.1 像素邻域

**4-邻域 (N₄(p))**：
```
  [ 0 1 0
    1 p 1
    0 1 0 ]
```

**对角邻域 (N_D(p))**：
```
  [ 1 0 1
    0 p 0
    1 0 1 ]
```

**8-邻域 (N₈(p))**：
```
  [ 1 1 1
    1 p 1
    1 1 1 ]
```

### 2.5.2 邻接、连通与区域

**邻接**：两个像素在空间上相邻

**连通**：
- **4-连通**：两像素是4-邻接且灰度值满足某条件
- **8-连通**：两像素是8-邻接且灰度值满足某条件
- **m-连通**：混合邻接，消除8-连通的歧义

**区域**：具有连通性的像素集合

### 2.5.3 距离度量

**欧氏距离 (Dₑ)**：
```
Dₑ(p, q) = √[(x₁-x₂)² + (y₁-y₂)²]
```

**城市街区距离 (D₄)**：
```
D₄(p, q) = |x₁-x₂| + |y₁-y₂|
```

**棋盘距离 (D₈)**：
```
D₈(p, q) = max(|x₁-x₂|, |y₁-y₂|)
```

## 2.6 灰度直方图（★★★☆☆）

### 6.1 定义

灰度直方图是图像中各灰度级出现的频率分布：

```
h(rₖ) = nₖ

其中：
  rₖ = 第 k 个灰度级
  nₖ = 灰度级为 rₖ 的像素个数
```

**归一化直方图**：
```
p(rₖ) = nₖ / M × N
```

### 6.2 直方图的应用

- **对比度分析**：观察灰度分布范围
- **阈值选择**：寻找双峰间的谷值
- **图像增强**：直方图均衡化、直方图匹配

### 6.3 直方图均衡化

**目的**：使输出图像的直方图趋于均匀分布，从而增强对比度。

**步骤**：
1. 计算输入图像的直方图 p(rₖ)
2. 计算累积分布函数 (CDF)
3. 建立变换函数：sₖ = T(rₖ) = Σ(p(rⱼ))
4. 根据变换函数重新映射灰度级

## 2.7 彩色图像基础

### 2.7.1 颜色模型

**RGB模型**：
- 加色模型
- 三通道：红、绿、蓝
- 用于显示器、摄像机

**CMY模型**：
- 减色模型
- 三通道：青、品红、黄
- 用于彩色打印

**HSI模型**：
- 更符合人眼感知
- H：色调 (Hue)
- S：饱和度 (Saturation)
- I：强度 (Intensity)
