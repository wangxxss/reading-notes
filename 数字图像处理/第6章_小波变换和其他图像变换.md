# 第6章 小波变换和其他图像变换

## 学习目标

- 理解从傅里叶变换到小波变换的发展历程
- 掌握多分辨率分析的基本概念
- 理解一维和二维离散小波变换
- 了解其他常用图像变换

---

## 6.1 背景介绍

### 6.1.1 傅里叶变换的局限性

**全局性**：
- 傅里叶变换将信号分解为全局正弦/余弦分量
- 无法同时提供时间和频率的局部化信息
- 适合平稳信号，不适合非平稳信号

**例子**：
- 一张包含边缘的图像：边缘位置信息丢失
- 时变信号：突变时刻无法定位

### 6.1.2 短时傅里叶变换 (STFT)

**思想**：将信号划分为短时段，对每段进行傅里叶分析

```
STFT{u,v} = ∫ f(t) × g*(t-τ) × e^(-j2πft) dt
```

**问题**：
- 窗口大小固定
- 时间分辨率和频率分辨率不可兼得

### 6.1.3 小波变换的引入

**核心思想**：
- 使用可变窗口
- 低频：大窗口（好频率分辨率）
- 高频：小窗口（好时间分辨率）
- 更符合人眼视觉特性

---

## 6.2 多分辨率分析基础

### 6.2.1 基本概念

**小波 (Wavelet)**：小型波，具有有限持续时间和振荡特性

**小波函数 ψ(t)**：
- ∫ψ(t)dt = 0（零均值）
- 能量集中
- 振荡性

**尺度函数 φ(t)**：
- 用于近似表示信号的低频部分

### 6.2.2 连续小波变换 (CWT)

**定义**：
```
CWT(a,b) = (1/√|a|) ∫ f(t) × ψ*((t-b)/a) dt
```

**参数**：
- a = 尺度参数（控制频率）
- b = 平移参数（控制位置）

### 6.2.3 离散小波变换 (DWT)

**定义**：
```
ψ_{j,k}(t) = 2^(j/2) × ψ(2^j t - k)
```

** Mallat 算法**：
- 使用滤波器组实现
- 高效计算

---

## 6.3 一维小波变换

### 6.3.1 小波级数展开

**信号展开**：
```
f(t) = Σ c_j0(k) × φ_{j0,k}(t) + Σ Σ d_j(k) × ψ_{j,k}(t)
               k               j≥j0 k
```

**系数**：
- c = 尺度系数（近似分量）
- d = 小波系数（细节分量）

### 6.3.2 快速小波变换 (FWT)

**分解过程**（使用Mallat算法）：
```
信号 → [低通滤波器 g] → 近似系数 c_{j-1}
     → [高通滤波器 h] → 细节系数 d_{j-1}
```

**重构过程**：
```
c_j = [上采样] → [低通滤波器 g'] → c_{j+1}
                         ↓
d_j = [上采样] → [高通滤波器 h'] → +
```

### 6.3.3 滤波器组关系

| 分解滤波器 | 重构滤波器 |
|------------|------------|
| g (低通) | g' (重建) |
| h (高通) | h' (重建) |

**正交小波条件**：
```
g' = g 的时间反转
h' = h 的时间反转
```

## 6.4 二维小波变换

### 6.4.1 可分离尺度函数

**二维尺度函数**：
```
φ(x,y) = φ(x) × φ(y)
```

### 6.4.2 可分离小波函数

**水平细节**：
```
ψ^H(x,y) = ψ(x) × φ(y)
```

**垂直细节**：
```
ψ^V(x,y) = φ(x) × ψ(y)
```

**对角细节**：
```
ψ^D(x,y) = ψ(x) × ψ(y)
```

### 6.4.3 二维DWT分解

```
┌─────────────────────────────────────────────┐
│                 原始图像                      │
│                  f(x,y)                      │
└─────────────────────────────────────────────┘
                      ↓
┌─────────────────────────────────────────────┐
│    LL (近似)  │    LH (水平)                │
│    ──────────┼─────────────                │
│    HL (垂直)  │    HH (对角)                │
└─────────────────────────────────────────────┘
```

**分解输出**：
- **LL**：低频近似，原图像的缩略图
- **LH**：水平边缘
- **HL**：垂直边缘
- **HH**：对角边缘

### 6.4.4 多级分解

```
Level 1:  [ LL1 │ LH1 ]
          ─────┼──────
          [ HL1 │ HH1 ]

Level 2:  [ LL2 │ LH2 │    │    ]
          ─────┼─────┤    │
          [ HL2 │ HH2 │ LH1│
                ─────┼──────
                [ HL1 │ HH1 ]
```

## 6.5 小波变换在图像处理中的应用

### 6.5.1 图像压缩

**优点**：
- 能量压缩效果好
- 多分辨率表示
- 适合渐进传输

**方法**：
1. 小波变换
2. 量化
3. 熵编码

### 6.5.2 图像去噪

**步骤**：
1. 进行小波变换
2. 对系数进行阈值处理
3. 重构图像

**阈值方法**：
- 硬阈值：保留大于阈值的系数
- 软阈值：收缩大于阈值的系数

### 6.5.3 边缘检测

**方法**：
1. 多级小波分解
2. 检测各子带中的局部极大值
3. 组合形成边缘

## 6.6 其他图像变换

### 6.6.1 离散余弦变换 (DCT)

**定义**：
```
C(u,v) = α(u)α(v) Σ Σ f(x,y) cos[π(2x+1)u/2N] cos[π(2y+1)v/2N]
         x   y
```

**特点**：
- 能量压缩效果好
- 虚部少（比DFT）
- 广泛应用：JPEG

### 6.6.2 Walsh-Hadamard变换

**特点**：
- 只使用 +1 和 -1
- 计算简单快速
- 无乘法运算

### 6.6.3 Radon变换

**定义**：
```
g(ρ,θ) = ∫∫ f(x,y) δ(ρ - x cosθ - y sinθ) dxdy
```

**应用**：
- CT重建
- 直线检测（Hough变换相关）

### 6.6.4 霍特林变换（PCA/KLT）

**目的**：去相关、特征提取

**步骤**：
1. 计算协方差矩阵
2. 求特征值和特征向量
3. 变换到新坐标系

## 6.7 小波包

### 6.7.1 概念

**小波包**：比小波更精细的分解

**分解**：
- 不仅分解低频
- 也分解高频
- 自适应选择最优基

### 6.7.2 小波包分解

```
原始信号
    ↓
[低频] ──── [高频]
  ↓            ↓
[LL][HL]   [LH][HH]
  ↓            ↓
 ...          ...
```

## 本章小结

### 变换方法对比

| 变换 | 特点 | 应用 |
|------|------|------|
| 傅里叶 | 全局频率分析 | 通用频率分析 |
| 短时傅里叶 | 固定窗口 | 时频分析 |
| 小波 | 多分辨率 | 图像压缩、去噪 |
| DCT | 能量压缩 | JPEG压缩 |
| PCA | 去相关 | 特征提取 |

### 二维小波分解结构

```
┌─────────────┬─────────────┐
│     LL      │     LH      │
│   (低频)    │ (水平细节)  │
├─────────────┼─────────────┤
│     HL      │     HH      │
│  (垂直细节)  │ (对角细节)  │
└─────────────┴─────────────┘
```

### 小波变换优势

1. **多分辨率**：适应不同频率成分
2. **局部化**：时频局部化
3. **计算效率**：Mallat算法
4. **稀疏表示**：图像能量集中
