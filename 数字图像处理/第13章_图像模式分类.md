## 13.2 原型匹配分类

### 13.2.1 基本原理

**核心思想**：为每个模式类预先定义原型，通过计算相似度进行分类

### 13.2.2 最小距离分类器

**原型定义**：
```
c_i = (1/N_i) Σ x
       x∈ω_i
```

**分类准则**：
```
x ∈ ω_i  if ||x - c_i|| < ||x - c_j|| for all j ≠ i
```

**决策边界**：
```
d_i(x) - d_j(x) = 0
即两类原型的垂直平分面
```

### 13.2.3 基于相关的匹配

**相关系数**：
```
r = [Σ (x - μₓ)(y - μᵧ)] / [√Σ(x - μₓ)² × √Σ(y - μᵧ)²]
```

**相关匹配**：
```
R(x,y) = Σ Σ f(s,t) w(s-x, t-y)
```

### 13.2.4 SIFT特征匹配

**特点**：
- 尺度不变
- 旋转不变
- 光照不变

**匹配过程**：
```
1. 提取关键点描述子
2. 在目标图像中搜索匹配
3. 使用最近邻匹配
4. 去除错误匹配(RANSAC)
```

## 13.3 最佳距离度量

### 13.3.1 马氏距离

```
d(x, μ) = √[(x - μ)ᵀ Σ⁻¹ (x - μ)]
```

**特点**：
- 考虑特征相关性
- 缩放不变
- 对协方差矩阵敏感

### 13.3.2 距离度量选择

| 度量 | 适用场景 |
|------|----------|
| 欧氏距离 | 特征独立、同方差 |
| 马氏距离 | 特征相关、不同方差 |
| 曼哈顿距离 | 对异常值鲁棒 |

## 13.4 支持向量机 (SVM)

### 13.4.1 基本原理

**最大间隔分类器**：
```
找到最大化两类间隔的超平面
```

**支持向量**：距离超平面最近的点

### 13.4.2 线性SVM

**决策函数**：
```
f(x) = wᵀx + b
```

**优化目标**：
```
min (1/2)||w||²
subject to: y_i(wᵀx_i + b) ≥ 1
```

### 13.4.3 核函数

**核技巧**：将数据映射到高维空间

| 核函数 | 公式 |
|--------|------|
| 线性 | K(x, y) = xᵀy |
| 多项式 | K(x, y) = (xᵀy + c)ᵈ |
| 高斯RBF | K(x, y) = exp(-||x - y||²/2σ²) |

### 13.4.4 多类SVM

| 方法 | 原理 |
|------|------|
| 一对多 (OvA) | N个二分类器 |
| 一对一 (OvO) | N(N-1)/2个分类器 |

## 13.5 神经网络与深度学习

### 13.5.1 神经网络基础

**神经元模型**：
```
y = f(Σ wᵢxᵢ + b)
```

**激活函数**：
| 函数 | 公式 |
|------|------|
| Sigmoid | 1/(1 + e⁻ˣ) |
| Tanh | (eˣ - e⁻ˣ)/(eˣ + e⁻ˣ) |
| ReLU | max(0, x) |

### 13.5.2 卷积神经网络 (CNN)

**基本结构**：
```
┌─────────────────────────────────────────────────┐
│  输入层 → 卷积层 → 池化层 → 全连接层 → 输出层   │
└─────────────────────────────────────────────────┘
```

**卷积层**：
```
- 局部连接
- 权重共享
- 提取局部特征
```

**池化层**：
```
- 最大池化
- 平均池化
- 减少维度
```

### 13.5.3 CNN在图像处理中的应用

| 应用 | 网络 |
|------|------|
| 图像分类 | AlexNet, VGG, ResNet |
| 目标检测 | R-CNN, YOLO, SSD |
| 语义分割 | FCN, U-Net |
| 人脸识别 | FaceNet, ArcFace |

### 13.5.4 深度神经网络的优势

| 优势 | 说明 |
|------|------|
| 特征自学习 | 自动学习层次化特征 |
| 泛化能力强 | 大数据训练 |
| 端到端 | 直接从输入到输出 |

## 13.7 分类器性能评价

### 13.7.1 混淆矩阵

|  | 预测正 | 预测负 |
|--|--------|--------|
| **实际正** | TP | FN |
| **实际负** | FP | TN |

### 13.7.2 性能指标

| 指标 | 公式 | 说明 |
|------|------|------|
| 准确率 | (TP+TN)/(TP+TN+FP+FN) | 整体正确率 |
| 精确率 | TP/(TP+FP) | 预测正类的正确率 |
| 召回率 | TP/(TP+FN) | 实际正类的识别率 |
| F1分数 | 2PR/(P+R) | 精确率和召回率的调和 |

### 13.7.3 ROC曲线

**ROC** = Receiver Operating Characteristic

**横轴**：假阳性率 (FPR)
**纵轴**：真阳性率 (TPR)

**AUC** = ROC曲线下面积，越大越好

### 13.7.4 交叉验证

**K折交叉验证**：
```
1. 将数据分为K份
2. 轮流用K-1份训练，1份测试
3. 平均K次结果
```
