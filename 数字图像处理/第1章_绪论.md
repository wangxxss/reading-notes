## 1.1 什么是数字图像处理

### 基本概念

**数字图像**：一幅图像可定义为一个二维函数 f(x, y)，其中 x 和 y 是空间（平面）坐标，而在任何一对空间坐标 (x, y) 处的幅值对应为函数值 f(x, y)，该幅值称为图像在该点处的**强度**或**灰度级**。当 x, y 和灰度值 f 是有限的离散数值时，我们称该图像为**数字图像**。

**数字图像处理**：通过数字计算机对输入的图像进行处理，使其在某方面的特性得到优化，使得：
- 更易于人眼观察
- 便于存储和传输
- 便于机器视觉分析

### 图像处理与其他相关领域的关系

| 领域 | 定义 |
|------|------|
| **计算机图形学** | 使用计算机创建图像 |
| **图像处理** | 通过数字计算机处理图像 |
| **计算机视觉** | 使计算机能够"理解"图像内容 |

## 1.2 数字图像处理的起源

### 历史发展

1. **1920年代**：报纸业使用电缆传输图像
2. **1960年代**：太空探索推动数字图像处理发展
3. **1970年代**：CT发明（诺贝尔奖），医学成像突破
4. **1980-1990年代**：个人计算机发展，图像处理技术普及
5. **2000年代至今**：深度学习崛起，图像处理与人工智能深度融合

- 深度神经网络 (Deep Neural Networks)
- 卷积神经网络 (CNN)

## 1.3 数字图像处理技术应用领域实例

### 1.3.1 伽马射线成像

- **医学诊断**：PET (正电子发射断层扫描)
- **天文观测**：探测高能宇宙射线

### 1.3.2 X射线成像

- **医学X射线**：胸片、牙科检查
- **工业检测**：无损检测、材料分析
- **安检系统**：行李扫描

### 1.3.3 紫外波段成像

- **荧光显微镜**：生物样品分析
- **天文观测**：紫外波段天体研究
- **工业应用**：UV固化、检测

### 1.3.4 可见光及红外波段成像

**可见光图像**：
- 彩色摄影
- 工业检测
- 文件扫描

**红外图像**：
- 遥感卫星
- 夜视仪
- 温度检测
- 农业监测

### 1.3.5 微波波段成像

- **雷达**：天气观测、地形测绘
- **合成孔径雷达 (SAR)**：高分辨率地面成像

### 1.3.6 无线电波成像

- **医学MRI**：磁共振成像
- **天文学**：射电望远镜

## 1.4 数字图像处理的基本步骤

```
图像获取 → 图像增强 → 图像复原 → 彩色图像处理
    ↓
小波变换 → 压缩 → 形态学处理 → 分割 → 特征提取
    ↓
图像识别 → 目标识别 → 图像理解
```

### 各步骤简介

| 步骤 | 说明 |
|------|------|
| **图像获取** | 通过传感器采集图像并数字化 |
| **图像增强** | 突出图像中感兴趣的特征 |
| **图像复原** | 恢复退化图像的质量 |
| **彩色图像处理** | 处理彩色图像和多通道图像 |
| **小波变换** | 多分辨率图像表示 |
| **压缩** | 减少图像存储空间 |
| **形态学处理** | 提取图像结构信息 |
| **分割** | 将图像划分为不同区域 |
| **特征提取** | 提取图像特征用于识别 |
| **图像分类** | 根据特征对图像进行分类 |

## 1.5 图像处理系统的组成

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│   采集设备   │ →  │   处理设备   │ →  │   显示/存储  │
│  (传感器)   │    │   (计算机)   │    │   (显示器)  │
└─────────────┘    └─────────────┘    └─────────────┘
```

### 主要组件

1. **图像采集模块**
   - 镜头
   - 传感器 (CCD/CMOS)
   - A/D 转换器

2. **处理模块**
   - 专用图像处理器
   - 通用计算机
   - GPU 加速

3. **存储模块**
   - 硬盘
   - 固态存储
   - 云存储

4. **显示模块**
   - 专业显示器
   - 打印输出
