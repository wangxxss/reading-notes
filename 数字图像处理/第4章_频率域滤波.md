# 第4章 频率域滤波

## 学习目标

- 理解傅里叶变换的基本原理
- 掌握频率域滤波的步骤
- 理解低通、高通滤波器的工作原理
- 能够设计和应用频率域滤波器

---

## 4.1 基本概念

### 4.1.1 频率域与空间域的关系

**空间域**：图像像素值随位置变化的域

**频率域**：图像灰度值变化的频率成分

```
┌─────────────────────────────────────────────┐
│                                             │
│   空间域 ──[傅里叶变换]──→ 频率域           │
│      ↑                        │             │
│      │                        ↓             │
│      └────[傅里叶逆变换]──────┘             │
│                                             │
└─────────────────────────────────────────────┘
```

### 4.1.2 频率分量的物理意义

| 分量 | 对应内容 |
|------|----------|
| **直流分量** (0频率) | 图像平均灰度 |
| **低频分量** | 缓慢变化的灰度（背景、大面积区域） |
| **高频分量** | 快速变化的灰度（边缘、噪声、细节） |

**关键洞察**：
- 90%的能量集中在1%的面积（低频区域）
- 高频部分决定图像的细节信息

### 4.1.3 频率域滤波的优势

1. **理论分析更方便**：卷积定理将空间滤波映射为频率域乘法
2. **特定频率处理**：可以精确控制需要增强或抑制的频率
3. **计算效率**：FFT算法使大图像处理更高效

## 4.2 傅里叶变换基础

### 4.2.1 一维傅里叶变换

**正向变换**：
```
F(u) = ∫ f(x) e^(-j2πux) dx
```

**逆变换**：
```
f(x) = ∫ F(u) e^(j2πux) du
```

### 4.2.2 二维傅里叶变换

**正向变换 (DFT)**：
```
F(u,v) = Σ Σ f(x,y) e^(-j2π(ux/M + vy/N))
         x   y
```

**逆变换 (IDFT)**：
```
f(x,y) = (1/MN) Σ Σ F(u,v) e^(j2π(ux/M + vy/N))
              u   v
```

**欧拉公式**：
```
e^(jθ) = cos(θ) + j sin(θ)
```

### 4.2.3 极坐标表示

```
F(u,v) = |F(u,v)| e^(jφ(u,v))

其中：
  |F(u,v)| = 幅值谱 = √(R² + I²)
  φ(u,v) = 相位谱 = arctan(I/R)
  R, I 分别是实部和虚部
```

### 4.2.4 傅里叶变换的性质

| 性质 | 说明 |
|------|------|
| **可分离性** | 二维DFT可分解为两个一维DFT |
| **平移性** | 空间域平移 → 频率域相位变化，幅值不变 |
| **旋转性** | 空间域旋转 → 频率域同角度旋转 |
| **周期性** | 频率域以 M×N 为周期重复 |
| **共轭对称** | F*(u,v) = F(-u,-v) → 幅值谱对称 |

**重要性质：线性**
```
F{af₁ + bf₂} = aF₁ + bF₂
```

## 4.3 频率域滤波基本步骤

```
┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐
│ 输入图像  │ → │ 乘以(-1) │ → │   DFT    │ → │ 滤波器   │ → │  IDFT    │ → │ 输出图像 │
│ f(x,y)   │   │  x+y     │   │  F(u,v)  │   │ H(u,v)   │   │          │   │  g(x,y)  │
└──────────┘   └──────────┘   └──────────┘   └──────────┘   └──────────┘   └──────────┘
                      ↓                                         ↓
              中心化处理                               乘以(-1)^(x+y)
```

### 滤波流程

1. **预处理**：f(x,y) × (-1)^(x+y) （将直流分量移至中心）
2. **正变换**：计算 F(u,v)
3. **滤波**：G(u,v) = H(u,v) × F(u,v)
4. **逆变换**：计算 g'(x,y)
5. **后处理**：g(x,y) = g'(x,y) × (-1)^(x+y)
6. **取实部**：忽略由于舍入误差产生的虚部

### 滤波模型

```
空间域: g(x,y) = h(x,y) * f(x,y)
频率域: G(u,v) = H(u,v) × F(u,v)

其中 H(u,v) 称为滤波器传递函数
```

## 4.4 低通滤波器

### 4.4.1 概念

**低通滤波器**：通过低频，阻止高频

**理想低通滤波器 (ILPF)**：
```
H(u,v) = 1  if D(u,v) ≤ D₀
       = 0  if D(u,v) > D₀

其中 D(u,v) = √[(u-M/2)² + (v-N/2)²]
     D₀ = 截止频率
```

**特点**：
- 理论上完美截止
- **严重振铃效应**（Gibbs现象）
- 不实际可用

### 4.4.2 巴特沃斯低通滤波器 (BLPF)

**n阶巴特沃斯低通滤波器**：
```
H(u,v) = 1 / [1 + (D(u,v)/D₀)^(2n)]
```

**特点**：
- 过渡带平滑
- 无锐利截止
- 振铃效应小于理想滤波器
- n 越大越接近理想滤波器

### 4.4.3 高斯低通滤波器 (GLPF)

```
H(u,v) = e^(-D²(u,v) / 2D₀²)
```

**特点**：
- 完全无振铃
- 时频域都是高斯函数
- 工程应用首选

### 低通滤波应用

- **图像模糊**：减少噪声、去除细节
- **预处理**：为其他处理准备图像
- **降噪**：抑制高频噪声

## 4.5 高通滤波器

### 4.5.1 概念

**高通滤波器**：通过高频，阻止低频

**作用**：
- 增强边缘
- 提取细节
- 图像锐化

### 4.5.2 理想高通滤波器 (IHPF)

```
H(u,v) = 0  if D(u,v) ≤ D₀
       = 1  if D(u,v) > D₀
```

### 4.5.3 巴特沃斯高通滤波器 (BHPF)

```
H(u,v) = 1 / [1 + (D₀/D(u,v))^(2n)]
```

### 4.5.4 高斯高通滤波器 (GHPF)

```
H(u,v) = 1 - e^(-D²(u,v) / 2D₀²)
```

### 4.5.5 高通滤波器与低通滤波器的关系

```
高通滤波器 = 1 - 低通滤波器
H_HP(u,v) = 1 - H_LP(u,v)
```

## 4.6 带通与带阻滤波器

### 4.6.1 带阻滤波器 (BRF)

**阻止**指定频率范围的滤波器

```
H(u,v) = 0  if D₀ - W/2 ≤ D(u,v) ≤ D₀ + W/2
       = 1  otherwise

W = 带宽
```

### 4.6.2 带通滤波器 (BPF)

**通过**指定频率范围的滤波器

```
H_BP(u,v) = 1 - H_BR(u,v)
```

### 应用

- **陷波滤波**：去除周期性噪声
- **选择性滤波**：处理特定频率干扰

## 4.7 频率域滤波实现细节

### 4.7.1 边界处理

| 方法 | 说明 |
|------|------|
| 零填充 | 简单但可能产生边缘效应 |
| 镜像延拓 | 更自然，但计算复杂 |
| 周期延拓 | 理论一致，实际常用 |

### 4.7.2 计算效率

**FFT算法复杂度**：
- 直接计算：O(M²N²)
- FFT计算：O(MN log(MN))

**二维DFT可分离性**：
```
F(u,v) = Σ [ Σ f(x,y) e^(-j2πux/M) ] e^(-j2πvy/N)
             x         y
```

可分解为行变换 + 列变换

## 4.8 频率域与空间域的联系

### 卷积定理

```
空间域卷积  ⟷  频率域乘法
f * h       ⟷  F × H
```

**意义**：空间域滤波器和频率域滤波器是等价的

### 对应关系

| 空间域 | 频率域 |
|--------|--------|
| 盒子函数 | sinc函数 |
| 高斯函数 | 高斯函数 |
| 冲击函数 | 常数 |

## 本章小结

### 频率域滤波器分类

```
┌─────────────────────────────────────────────┐
│              频率域滤波器                    │
├─────────────┬─────────────┬─────────────────┤
│   低通       │   高通      │   带通/带阻      │
├─────────────┼─────────────┼─────────────────┤
│ 理想 (ILPF)  │ 理想 (IHPF) │ 理想 (IBRF/IBPF) │
│ 巴特沃斯     │ 巴特沃斯     │ 巴特沃斯          │
│ 高斯 (GLPF)  │ 高斯 (GHPF) │ 高斯              │
└─────────────┴─────────────┴─────────────────┘
```

### 滤波器特性对比

| 滤波器 | 振铃 | 过渡带 | 计算复杂度 |
|--------|------|--------|-----------|
| 理想 | 严重 | 锐利 | 低 |
| 巴特沃斯 | 轻 | 平滑 | 中 |
| 高斯 | 无 | 平滑 | 低 |

### 频率域滤波步骤

1. 乘以 (-1)^(x+y) 中心化
2. 计算 DFT
3. 乘以 H(u,v) 滤波
4. 计算 IDFT
5. 乘以 (-1)^(x+y) 反中心化
6. 取实部
