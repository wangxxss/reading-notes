**知识体系概览：**

```
敏捷开发
├── 价值观层：敏捷宣言（4条）
├── 原则层：敏捷12条原则 + SOLID 5大原则
└── 实践层：XP极限编程实践 + 设计模式 + 重构
```

**核心主题：**
- 敏捷宣言与12条原则
- 极限编程(XP)十二项实践：TDD、结对编程、持续集成、重构等
- SOLID五大设计原则：SRP、OCP、LSP、ISP、DIP
- 创建型/结构型/行为型设计模式
- 包的设计原则(REP、CRP、CNP、ADP、SDP、SAP)
- 典型案例：薪水支付系统
